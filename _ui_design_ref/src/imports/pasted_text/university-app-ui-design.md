Create a complete mobile app UI design for a University application called "University Students".

The app supports 4 user roles with different dashboards and permissions:
1. Admin (Dean / Student Office)
2. Club Lead
3. Faculty
4. Student

Design a modern, clean, student-friendly mobile UI with role-based access and engaging layouts.

Design Theme:
- Professional university style
- Primary color: #276C84
- Light background: #F8FAFC
- Rounded cards and soft shadows
- Clear icons and structured layout
- Modern typography (Inter or Poppins)
- Consistent spacing and hierarchy
- Card-based interface
- Mobile-first responsive design

Create separate UI screens for each role.

UI STYLE GUIDELINES

- Use rounded corners (12–16px)
- Use soft drop shadows
- Maintain consistent spacing (16px / 24px)
- Use card-based layout
- Use clear section headers
- Use icons for actions
- Keep layout clean and modern
- Ensure role-based dashboards look different but consistent
- Make interface engaging and professional for university use

Create all screens as separate mobile frames with consistent navigation and layout.

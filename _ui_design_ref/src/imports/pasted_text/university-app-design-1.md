Design a complete mobile application UI and wireframe system for a university platform called:

"University Students"

The app is a centralized university management system that connects Admin (Dean / Student Office), Club Lead, Faculty, and Student users.

Assume no existing wireframe or design. Create the full structure from the beginning.

Design Type:
Mobile Application
Modern UI
Clean and professional
Student-friendly
Role-based dashboards
Card-based layout
Minimal but engaging design

Use grayscale wireframe first, then apply modern UI styling.

Create separate mobile frames for each screen.

-----------------------------------------------------

USER ROLES IN THE SYSTEM

1. Admin
- Department Dean
- Student Office Staff

2. Club Lead
- Head of a student club

3. Faculty
- Teachers and professors

4. Student
- University students

Each role must have its own dashboard and permissions.
